01
Landing Page
marketing · arc-style · animated on scroll
🌊
Background orbs3 radial gradient blobs behind the hero. Animate with CSS: @keyframes orbFloat — translateY(-30px) + scale(1.04), duration 8–12s, direction: alternate. One indigo, one purple, one mint.
⌨️
Live code typing in mockupLine 5 of the editor mock types itself character by character. Implement with setInterval every 80ms, appending one char. Stop when line is complete.
🖱️
Collaboration cursors driftTwo named cursors (Prof. Chen + Jordan) drift slightly every 1.8s via setInterval with Math.random() ±2px on x, ±1px on y. Use CSS transition: left 0.6s ease to smooth movement.
📜
Scroll-reveal sectionsEvery section below the hero uses IntersectionObserver (threshold: 0.12). On intersection: opacity: 0→1 + translateY(28px→0) over 0.7s ease. Stagger children 100ms apart.
🌀
Scroll indicatorMouse icon at bottom-center. Inner dot animates: translateY(0→12px) + opacity 1→0, 1.6s ease-in-out infinite loop. Hides when user scrolls past hero.
🔴
Live dot pulseThe mint green dot in the pill badge uses @keyframes pulseDot — scale(1→0.85) + opacity(1→0.6), 2s infinite. Used everywhere a "live" indicator appears.
02
Teacher Dashboard
teacher role · overview of all classrooms
📊
Stats cards4-column grid. Each card: white bg, 1px border, borderRadius 16px, padding 22px. Icon in a 36×36 rounded-10 box with color at 18% opacity. Value in 32px/800 weight. No animation — static display.
🏫
Classroom progress barsEach classroom card has a 4px progress bar at the bottom: active / total students × 100%. Fill color matches the classroom's accent color. Animate fill width with transition: width 0.6s ease on mount.
🟢
Card hover stateOn :hover: border-color changes to var(--indigo), box-shadow appears. Use transition: all 0.14s. Cards with onClick get cursor: pointer.
03
Student Dashboard
student role · Steam-like profile · gamification
🎮
Profile hero gradientBackground is user-selected from cosmetics shop. Default: linear-gradient(135deg, oklch(0.52 0.26 265), oklch(0.48 0.27 265)). Overlay: CSS grid pattern at rgba(255,255,255,0.03), 40px cells.
💍
Avatar frame glowThe avatar border color + box-shadow glow both come from the equipped frame cosmetic. Gold frame: border: 3px solid #ffd700; box-shadow: 0 0 16px #ffd70066. Animate with transition: box-shadow 0.4s, border-color 0.4s when equipping.
⬡
XP progress barCustom component, NOT a native range input. Height 6px, bg rgba(255,255,255,0.15), fill color white with box-shadow: 0 0 8px white. Animate fill width from 0→actual on mount with transition: width 1s ease.
🛒
Cosmetics shop toggle"✦ Customize" button reveals shop panel with animation: fadeIn 0.2s ease. Clicking a frame tile immediately updates the avatar preview above. Locked items show opacity: 0.5; cursor: not-allowed. Equipped item shows indigo border.
🏆
Achievement badges hoverUnlocked badges: transform: scale(1.05) on hover, transition 0.15s. Locked badges: opacity: 0.45; filter: grayscale(1). Rarity colors: common=#94a3b8, rare=#818cf8, epic=purple, legendary=#fbbf24.
📅
Activity heatmap15 weeks × 7 days grid. Cell size 10×10px, gap 3px, borderRadius 2px. 5 color tiers: transparent, #1e3a5f, #1d4ed8, #3b82f6, #818cf8. No animation — static render.

Classroom View
teacher & student role · tabs: Students | Assignments
🔴
Live student indicatorOnline students show a pulsing mint dot (see pulseDot keyframe) bottom-right of their avatar — 10px, border 1.5px solid bg color. Offline shows a static grey dot.
📋
Assignment progress bar4px bar below each assignment row. Fill = submissions / total × 100%. Indigo color. Animate width on mount with 0.6s ease transition.
🖱️
Row hover stateStudent/assignment rows: background: var(--bg-2) on hover, transition 0.12s. Rows are full-width clickable targets (cursor: pointer).
05
IDE Editor
student & teacher role · dark surface · always dark regardless of theme
🌳
Chapter file tree3-level tree: Chapters → Sessions → files. Chapters and sessions have a ▶ arrow that rotates 90° when open (transition: transform 0.15s). Active file: background: #1e2a3a. Worksheet.py has an inline "shared" tag badge.
👆
Collaboration cursorsAbsolute-positioned 2px × 18px colored bars on specific lines. Above each cursor: floating name label (9px/700, bg matches cursor color). Cursors blink with @keyframes blink 1.2s infinite. Each user has a unique color.
✏️
Syntax-highlighted editorA transparent <textarea> sits on top of a read-only <div> with HTML spans for colors. caretColor: #e2e8f0, color: transparent on the textarea. Scroll is synced between both layers.
🎉
Confetti on submit90 pieces from top of screen. Random x (0–100%), shapes: circle/square/rectangle, colors from 7-color palette. @keyframes confettiFall: translateY(105vh) + rotate(720deg) over 1.2–2.6s. Success popup card uses @keyframes popIn: scale(0.6→1.05→1) + opacity.
🖊️
Drawing board slide-inClick "✏ Draw" button → panel slides in from right: @keyframes slideInRight (translateX 100%→0), 0.2s ease. Has a fullscreen toggle. Canvas bg: #1a1f2e with CSS grid overlay. Drawing uses HTML5 Canvas API with lineCap/lineJoin: round.
▶
Run simulationOutput lines appear one by one every 300ms using setInterval. "running…" text pulses with @keyframes pulse in the terminal tab bar while executing.
📌
Right panel — role-basedTeacher sees: Students tab (who's editing + line number) + Submissions tab (all). Student sees: only their own Submissions tab. Expandable submission rows show inline test results on click.
06
Submission Review
teacher role · code viewer + test results
📋
Active row highlightSelected submission: background: var(--indigo-10); border-left: 3px solid var(--indigo). Name changes to indigo color. Transition: 0.12s background.
💻
Code displayDark surface (#0f1117), DM Mono 12.5px. Syntax highlighting via dangerouslySetInnerHTML (same tokenizer as editor). Line numbers in a left column at #1e2a3a color.
✅
Test result iconsPass: ✓ in var(--mint). Fail: ✗ in var(--text-3). Test name in DM Mono 12px. Score shown as large percentage, color: green if ≥80%, amber if >0%, rose if 0%.
